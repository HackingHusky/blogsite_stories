Automating Nmap with Python

Honestly, anything to make the OSCP easier on yourself. This is a quick grab-and-go, fairly easy to use. In Kali Linux, I had to learn a few steps to get it running with its libraries; however, I made it work.

Really go through readme.md on my GitHub repo:

GitHub - HackingHusky/python-nmap-scan: python script that does nmap scan and gives results
python script that does nmap scan and gives results - HackingHusky/python-nmap-scangithub.com

You want to make sure you run it like this:

Install Python dependencies

pip install -r requirements.txt

#or

pip install python-nmap

#in kali though:

or in kali

Step 1 — Make sure venv support is installed

sudo apt isntall python3-venc

Step 2 — Create a virtual environment

python3 -m venv nmap-env

Step 3 — Activate it

source nmap-env/bin/activate

Your prompt will change to something like:

(nmap-env)kali@kali~$:

Step 4 — Install python‑nmap normally

pip install python-nmap

After that, the command tags are simple:

python3 scan.py -t <target’s IP> -p (ports you want to look up) -a "nmap arguments

This ended up running a lot quicker on my own machine:

You can add the subnet and multiple IP addresses as well. It runs like nmap, but it's much quicker and returns results much faster than I expected.
